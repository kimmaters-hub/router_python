# Implement Router

A Python class that simulates a network router with memory limits, packet forwarding, and timestamp-based queries.

## Features
- FIFO packet forwarding
- Duplicate detection
- Memory limit enforcement with eviction
- Efficient timestamp range queries

## Files
- `implement_router.py`: Core Router class
- `demo.py`: Sample usage
- `tests/test_router.py`: Unit tests

## Run Demo
```bash
python demo.py
